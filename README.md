# 蛙設計稿 - 一頁式登入頁切版

依照設計稿需求切版，最小版面寬 1200px (網頁總寬 1920px)

## Demo

https://mengtingku.github.io/wow-login/

## 技術架構

-   React：構建網站的用戶介面。
-   Vite：作為開發伺服器，提供快速的即時更新。
-   scss：自定義樣式和主題。
-   RWD：需求方不用 UI 框架，所以改以自定義斷點處理。
-   Animate.css：套用現成動畫達到動態滑入需求。

## 安裝與使用

```bash
git clone https://github.com/MengtingKu/wow-login.git

cd wow-login

npm install

npm run dev
```
